# АТОЛ Чек-генератор

Windows-приложение для подготовки и контроля кассовых чеков АТОЛ Онлайн,
исправлений по ФФД 1.05 и обмена фискальными данными с 1С УТ 10.3.

Программа не заменяет обычное пробитие реализаций кассирами в 1С. Для реализаций
она используется прежде всего тогда, когда нужен чек коррекции или исправление
уже сформированного ошибочного чека.

## Что умеет программа

- оформлять возвраты по сообщениям бухгалтера через API АТОЛ Онлайн;
- формировать обычные чеки оплаты и редкий ручной чек реализации;
- загружать реализации из 1С и готовить XML чеков коррекции;
- вести список ошибок из Obsidian и открывать их в отдельном окне исправлений;
- создавать XML и служебные записки для исправительных сценариев;
- загружать журналы АТОЛ и отчёты ОФД/Такскома;
- искать отсутствующие чеки в личном кабинете Такском по ФПД;
- сопоставлять XML, АТОЛ и ОФД и готовить CSV для внешней обработки 1С;
- хранить кассиров, агентов, темы и пути к рабочим файлам в настройках;
- проверять наличие новой версии в GitHub Releases.

## Быстрый старт

1. Скачайте `AtolGenerator-x64.exe` из последнего GitHub Release. Вариант x86
   нужен для 32-разрядной Windows или 32-разрядного COM-коннектора 1С.
2. Откройте `Настройки` и заполните подключения АТОЛ и 1С, кассира, агентов и
   путь к файлу Obsidian.
3. Для обычных возвратов используйте `Возвраты по заказам`.
4. Для реализаций используйте `Коррекции реализаций`.
5. Для накопленных ошибок откройте `Исправление чеков`, подготовьте план и
   передайте выбранные строки в отдельное окно исправлений.

Результаты сохраняются рядом с программой в `atol_output/`.

## Сборка

Требуются Windows и .NET 8 SDK.

```powershell
dotnet restore
dotnet build AtolGenerator.csproj -c Debug
dotnet publish AtolGenerator.csproj -c Release -r win-x64 --self-contained true
```

Debug-файл: `bin/Debug/net8.0-windows/AtolGenerator.exe`.
