## Что считается источником истины

1. Для формата запросов АТОЛ - действующее описание API v4 и XSD/примеры XML
   на официальном сайте АТОЛ.
2. Для исправления существующего чека - официальная инструкция АТОЛ по ФФД 1.05.
3. Для факта фискализации - данные ОФД/Такскома и карточка конкретного чека.
4. Для документа, подразделения, суммы и номенклатуры - 1С.
5. Для незакрытых ошибок и пояснений - рабочий файл Obsidian.
6. Для фактически реализованного поведения программы - исходный код текущего тега.
